[Application and Process Management]

#include <wx/dynlib.h>

List of all members.

Detailed Description

This class is used for the objects returned by the wxDynamicLibrary::ListLoaded() method and contains the information about a single module loaded into the address space of the current process. A module in this context may be either a dynamic library or the main program itself.
