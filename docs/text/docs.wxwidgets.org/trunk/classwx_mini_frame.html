Detailed Description

A miniframe is a frame with a small title bar. It is suitable for floating toolbars that must not take up too much screen area.

An example of mini frame can be seen in the dialogs sample using the "Mini frame" command of the "Generic dialogs" submenu.

Styles:

  • wxICONIZE:
    Display the frame iconized (minimized) (Windows only).
  • wxCAPTION:
    Puts a caption on the frame.
  • wxMINIMIZE:
    Identical to wxICONIZE.
  • wxMINIMIZE_BOX:
    Displays a minimize box on the frame (Windows and Motif only).
  • wxMAXIMIZE:
    Displays the frame maximized (Windows only).
  • wxMAXIMIZE_BOX:
    Displays a maximize box on the frame (Windows and Motif only).
  • wxCLOSE_BOX:
    Displays a close box on the frame.
  • wxSTAY_ON_TOP:
    Stay on top of other windows (Windows only).
  • wxSYSTEM_MENU:
    Displays a system menu (Windows and Motif only).
  • wxTINY_CAPTION_HORIZ:
    This style is obsolete and not used any longer.
  • wxTINY_CAPTION_VERT:
    This style is obsolete and not used any longer.
  • wxRESIZE_BORDER:
    Displays a resizeable border around the window.
