Detailed Description

wxWidgets maintains a database of standard RGB colours for a predefined set of named colours. The application may add to this set if desired by using AddColour() and may use it to look up colours by names using Find() or find the names for the standard colour using FindName().

There is one predefined, global instance of this class called wxTheColourDatabase.
