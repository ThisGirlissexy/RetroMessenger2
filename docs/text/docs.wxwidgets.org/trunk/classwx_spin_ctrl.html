Detailed Description

wxSpinCtrl combines wxTextCtrl and wxSpinButton in one control.

Styles:

  • wxSP_ARROW_KEYS:
    The user can use arrow keys to change the value.
  • wxSP_WRAP:
    The value wraps at the minimum and maximum.
