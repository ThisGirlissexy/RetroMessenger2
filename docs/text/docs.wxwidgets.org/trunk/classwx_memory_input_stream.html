wxMemoryInputStream::wxMemoryInputStream ( const char *  data,
size_t  len 
)

Creates a new read-only memory stream, initializing it with the data from the given input stream stream. The len argument specifies the amount of data to read from the stream. Setting it to wxInvalidOffset means that the stream is to be read entirely (i.e. till the EOF is reached).
