Keycodes

Include file: 
#include <wx/defs.h>

Keypresses are represented by an enumerated type, wxKeyCode. The possible values are the ASCII character codes, plus the following:

    WXK_BACK    = 8
    WXK_TAB     = 9
    WXK_RETURN  = 13
    WXK_ESCAPE  = 27
    WXK_SPACE   = 32
    WXK_DELETE  = 127
