wxUint32 wxZipEntry::GetExternalAttributes (  ) 

The low 8 bits are always the DOS/Windows file attributes for this entry. The values of these attributes are given in the enumeration wxZipAttributes. The remaining bits can store platform specific permission bits or attributes, and their meaning depends on the value of systemmadeby() SetSystemMadeBy. If IsMadeByUnix() is true then the high 16 bits are unix mode bits. The following other accessors access these bits: wxArchiveEntry::isreadonly IsReadOnly/SetIsReadOnly

int wxZipEntry::GetMode (  ) 

Sets the DOS attributes in externalattributes() GetExternalAttributes to be consistent with the mode given. If IsMadeByUnix() is true then also stores mode in GetExternalAttributes(). Note that the default constructor sets systemmadeby() GetSystemMadeBy to wxZIP_SYSTEM_MSDOS by default. So to be able to store unix permissions when creating zips, call SetSystemMadeBy(wxZIP_SYSTEM_UNIX).

void wxZipEntry::SetNotifier ( wxZipNotifier &  notifier  ) 

Sets the notifier() for this entry. Whenever the wxZipInputStream updates this entry, it will then invoke the associated notifier's wxZipNotifier::OnEntryUpdated method. Setting a notifier is not usually necessary. It is used to handle certain cases when modifying an zip in a pipeline (i.e. between non-seekable streams).
