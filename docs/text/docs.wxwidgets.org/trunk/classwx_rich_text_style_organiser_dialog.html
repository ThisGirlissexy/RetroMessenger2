Detailed Description

This class shows a style sheet and allows the user to edit, add and remove styles. It can also be used as a style browser, for example if the application is not using a permanent wxRichTextStyleComboCtrl or wxRichTextStyleListCtrl to present styles.

Constructors. To create a dialog, pass a bitlist of flags (see below), a style sheet, a text control to apply a selected style to (or NULL), followed by the usual window parameters. To specify the operations available to the user, pass a combination of these values to flags:

wxRICHTEXT_ORGANISER_DELETE_STYLES

Provides a button for deleting styles.

wxRICHTEXT_ORGANISER_CREATE_STYLES

Provides buttons for creating styles.

wxRICHTEXT_ORGANISER_APPLY_STYLES

Provides a button for applying the currently selected style to the selection.

wxRICHTEXT_ORGANISER_EDIT_STYLES

Provides a button for editing styles.

wxRICHTEXT_ORGANISER_RENAME_STYLES

Provides a button for renaming styles.

wxRICHTEXT_ORGANISER_OK_CANCEL

Provides OK and Cancel buttons.

wxRICHTEXT_ORGANISER_RENUMBER

Provides a checkbox for specifying that the selection should be renumbered.

The following flags determine what will be displayed in the style list:

wxRICHTEXT_ORGANISER_SHOW_CHARACTER

Displays character styles only.

wxRICHTEXT_ORGANISER_SHOW_PARAGRAPH

Displays paragraph styles only.

wxRICHTEXT_ORGANISER_SHOW_LIST

Displays list styles only.

wxRICHTEXT_ORGANISER_SHOW_ALL

Displays all styles.

The following symbols define commonly-used combinations of flags:

wxRICHTEXT_ORGANISER_ORGANISE

Enable all style editing operations so the dialog behaves as a style organiser.

wxRICHTEXT_ORGANISER_BROWSE

Show a list of all styles and their previews, but only allow application of a style or cancellation of the dialog. This makes the dialog behave as a style browser.

wxRICHTEXT_ORGANISER_BROWSE_NUMBERING

Enables only list style browsing, plus a control to specify renumbering. This allows the dialog to be used for applying list styles to the selection.
