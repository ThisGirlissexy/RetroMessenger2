int wxFontDialog::ShowModal (  ) 

Shows the dialog, returning wxID_OK if the user pressed Ok, and wxID_CANCEL otherwise. If the user cancels the dialog (ShowModal returns wxID_CANCEL), no font will be created. If the user presses OK, a new wxFont will be created and stored in the font dialog's wxFontData structure.
