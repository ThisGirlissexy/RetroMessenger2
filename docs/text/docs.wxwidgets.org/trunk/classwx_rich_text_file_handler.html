Detailed Description

This is the base class for file handlers, for loading and/or saving content associated with a wxRichTextBuffer.

void wxRichTextFileHandler::SetFlags ( int  flags  ) 

Sets flags that change the behaviour of loading or saving. See the documentation for each handler class to see what flags are relevant for each handler. You call this function directly if you are using a file handler explicitly (without going through the text control or buffer LoadFile/SaveFile API). Or, you can call the control or buffer's SetHandlerFlags function to set the flags that will be used for subsequent load and save operations.
